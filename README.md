### Nikrex-это-это инструмент для SMS-спама :bomb:  (работает на termux и на linux)
## Установка
* apt update && apt upgrade
* pkg install python
* apt install git
* git clone https://github.com/nkolotovkinOfficial/Nikrex132.git
* cd nikrex
* pip3 install -r requirements.txt
## Запуск
* python3 nikrex.py
